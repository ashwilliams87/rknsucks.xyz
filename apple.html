Инструкция по настройке OpenVPN для iOS
Данная инструкция работает на iOS ≥ 8
1
Установите приложение OpenVPN Connect из AppStore.
2
Скачайте профиль автоконфигурации VPN.
rknsucks_public_client.ovpn
Важно: Используйте браузер Safari. При открытии ссылки через мессенджеры или из почты файл автоконфигурации может не работать.
3
Импортируйте файл автоконфигурации и подключитесь к VPN.
После подключения вы увидите значок VPN в области уведомлений вверху. Для управления подключением используйте программу OpenVPN Connect.

VPN активируется только для отдельных ресурсов, для которых прописана маршрутизация на сервере. Остальные сайты будут работать, как и раньше, напрямую через вашего провайдера.

Постоянное подключение к VPN может незначительно влиять на расход батареи.
Как удалить OpenVPN с устройства iOS
Для полного удаления OpenVPN с устройства удалите приложение OpenVPN Connect.
